title=Detailed changelog for Adblock Plus 1.3.2

{{s1 The following lists the changes compared to Adblock Plus 1.3.1.}}

  • {{s2 Persian translation added again, some other translations updated.}}
  • {{s3 Issue reporter: Page referrer is being sent to allow recognizing/investigating pop-up issues more easily.}}
  • {{s4 Issue reporter: Sending a report is no longer possible in some scenarios (prevents useless reports).}}
  • {{s5 Fixed: Element hiding rules stop working after they have been subject to an exception (bug 23257).}}
  • {{s6 Fixed: Toolbar icon's text shows up to the right of the icon rather than below it (forum topic).}}
  • {{s7 Fixed: Hit counts might not get saved when the browser is closed (Firefox 4 only).}}
  • {{s8 Fixed: Links in About dialog open in an external application on Linux (bug 23441).}}
  • {{s9 Fixed: List of recently reported issues has horizontal scrolling with some themes (forum topic).}}
  • {{s10 Fixed: Toolbar icon is broken in SeaMonkey Mail nightly builds.}}
  • {{s11 Fixed: Dragging a filter in Preferences might cause "unresponsive script" warning.}}
  • {{s12 Fixed: "Invalid XBL binding" warning in Firefox 4.}}
  • {{s13 Fixed: Adblock Plus is confusing memory leak detection tools, a bogus leak is being reported.}}
