title=Detailed changelog for Adblock Plus 2.1

{{s1 The following lists the major changes compared to Adblock Plus 2.0.3.}} {{s2 If you experience issues with this release please check the list of known issues.}}

{{s3 Features}}

  • {{s4 Adblock Plus can be installed without restarting the browser now.}}
  • {{s5 Added support for Firefox Mobile 14.}}
  • {{s6 Better Thunderbird integration, in particular the Adblock Plus user interface is working with web page tabs correctly now.}}
  • {{s7 Implemented element hiding exceptions (mostly useful for subscription authors).}}

{{s8 User interface}}

  • {{s9 Filter composer: Removed warning when adding filters to a disabled group, a new group is created if no enabled group can be found for the filter.}}
  • {{s10 Filter preferences: Various improvements for a better overview (forum topic, forum topic, forum topic).}}
  • {{s11 Filter preferences: Fixed wrong warning message when deleting filter groups.}}
  • {{s12 Issue reporter: Force a filter subscription update when reporting a false positive (forum topic).}}
  • {{s13 Issue reporter: Sending issue reports is no longer allowed if the filter subscription hasn't been updated in the past two weeks.}}
  • {{s14 Custom About dialog has been removed, using the browser's built-in dialog now.}}
  • {{s15 Fixed: Tabs on Flash and Java no longer have rounded corners in Firefox 13 (forum topic).}}

{{s16 Blockable items}}

  • {{s17 The shortcut key for Blockable items in Firefox 15 and above is Ctrl-Shift-U now, the old shortcut key is no longer available.}}
  • {{s18 Searches for}} object-subrequest {{s19 are accepted as well, searching for}} object_subrequest {{s20 is no longer required (forum topic).}}
  • {{s21 Fixed: The list will sometimes reload even though the current page stays the same.}}
  • {{s22 Fixed: Context menu might suggest disabling the filter on wrong domain (forum topic).}}
  • {{s23 Fixed: Context menu might prevent page elements from being garbage collected (bug 727739).}}
  • {{s24 Fixed: The tooltip might get displayed on top of the context menu.}}

{{s25 Core}}

  • {{s26 Adblock Plus can now be installed in any application based on Gecko 8.0 and higher, in particular Firefox 8, Thunderbird 8, SeaMonkey 2.5. Older versions are no longer supported, XUL-based Firefox Mobile releases like Firefox Mobile 10 aren't supported either.}}
  • {{s27 Filter subscriptions can set their title automatically using}} ! Title {{s28 comment (forum topic).}}
  • {{s29 Improved memory use and performance, especially fixed the regression in Firefox 15 and above.}}
  • {{s30 Stopped using}} cache.js {{s31 file, it's a frequent source of stability issues.}}
  • {{s32 Switched to asynchronous file read/write operations to improve user interface responsiveness.}}
  • {{s33 Fixed: Object tabs are causing "}}can't access dead object{{s34 " message in Error Console of Firefox 15 and above.}}
  • {{s35 Fixed handling of trailing dots in domain names (forum topic).}}
