title=Privacy Policy

{{s1 General notes}}

{{s2 The general privacy policy of the Adblock Plus project is to avoid collecting more data than is required for the extensions and the AdblockPlus.org website to function correctly.}} {{s3 The data gathered is anonymized if possible and removed when no longer required.}} {{s4 It is never shared with any third parties.}} {{s5 This policy lists the data that is collected and the processing applied to it.}}

{{s6 AdblockPlus.org website}}

{{s7 Website logs}}

{{s8 All requests to the AdblockPlus.org website are recorded in the website logs.}} {{s9 The data stored includes your IP address, time of the request, the web address accessed, the browser identifier and the referring page.}} {{s10 This data is used to generate usage statistics as well as to investigate potential security issues and forum/blog spam.}} {{s11 The detailed logs are retained for a period of 30 days after which only the aggregated usage statistics remain.}}

{{s12 Stored IP addresses}}

{{s13 The web applications installed on AdblockPlus.org store your IP address when you create a forum post, add a blog comment or translate a page.}} {{s14 This allows administrators to address spam content and security breaches effectively.}} {{s15 All stored IP addresses are removed after 30 days.}}

{{s16 Cookies}}

{{s17 Several areas of the Adblock Plus website use cookies to recognize logged in users or to store settings.}} {{s18 Cookies are not evaluated in any other manner and are never used to track website visitors.}}

{{s19 Forum registration}}

{{s20 Data entered during forum registration is used to form user's public profile and is therefore visible to other forum visitors.}} {{s21 There are two exceptions:}}

  • {{s22 Passwords go through one-way encryption before being stored on the server.}} {{s23 This means that they cannot be recovered by anybody, even with access to the database.}}
  • {{s24 Email addresses are only displayed in the public profile if explicitly agreed to.}} {{s25 By default they are only used for the sending of notifications you subscribed to.}}

{{s26 Registration is not required to participate in the forums.}}

{{s27 Blog comments}}

{{s28 When a comment is added on the Adblock Plus blog an email address can optionally be specified.}} {{s29 Unlike the other fields it is never displayed and is only used to notify the comment author about replies.}}

{{s30 Adblock Plus extension}}

{{s31 Extension update checks}}

{{s32 Your browser periodically checks for updates of all your extensions including Adblock Plus.}} {{s33 Some general information, such as browser version, extension version, operating system and your IP address, are transmitted during an update check.}} {{s34 Adblock Plus nightly build updates as well as updates for Adblock Plus on Android and Internet Explorer are handled by AdblockPlus.org website and the data transmitted is subject to its privacy policy.}} {{s35 Updates to stable Adblock Plus for Firefox releases are handled by Addons.Mozilla.Org website and are subject to the}} {{s36 Firefox Privacy Policy}}{{s37 .}} {{s38 Updates to stable Adblock Plus for Chrome releases are handled by the Google Web Store website and is subject to the Google Privacy Policy.}} {{s39 Updates to stable Adblock Plus for Opera releases are handled by the Opera Add-ons website and are subject to the}} {{s40 Opera privacy policy}}{{s41 .}}

{{s42 Subscription downloads}}

{{s43 If you add filter subscriptions to your Adblock Plus installation the subscription will be requested regularly to retrieve updates.}} {{s44 Every update results in the hosting website receiving your IP address as well as some general information like your Adblock Plus version, browser and browser version.}} {{s45 This data is subject to the privacy policy of the website in question.}}

{{s46 Furthermore, if a subscription download fails on several consecutive occasions, the updated address of the filter list is requested from the AdblockPlus.org domain.}} {{s47 The data transmitted includes Adblock Plus version, subscription address, and information about the error encountered.}} {{s48 This data is used to identify issues that have not been reported by subscription maintainers and is subject to the usual AdblockPlus.org privacy policy.}}

{{s49 Issue reporter}}

{{s50 Adblock Plus for Firefox allows you to send issue reports to be temporarily stored on AdblockPlus.org domain.}} {{s51 These reports contain information required to investigate the issue including Adblock Plus version, browser version, address of the web page where the problem is visible, blockable items on this web page, matching filters and active subscriptions.}} {{s52 Parameter values are removed from all transmitted addresses to avoid unintentionally leaking private information.}} {{s53 It is also possible to provide additional information, which may be privacy-sensitive:}}

  • {{s54 Website screenshot provides a scaled-down and color-reduced version of the page viewed.}} {{s55 It is possible to remove particularly sensitive sections of the page before the report is sent.}}
  • {{s56 Email address is only used to request more information or to update the user on the issue.}}
  • {{s57 List of installed extensions is necessary for the investigation of conflicts with other extensions.}}

{{s58 The processed reports can only be accessed by somebody who knows their unique address.}} {{s59 These addresses are only shared with the maintainers of the filter subscriptions mentioned in the report and are unlikely to be found by others unless a link is provided.}} {{s60 All reports and their associated data are automatically removed after 30 days.}}

{{s61 Emergency notification}}

{{s62 Recent Adblock Plus versions contain an emergency notification mechanism that allows them to notify users about important issues affecting their Adblock Plus installation.}} {{s63 This feature requires all Adblock Plus installations to query an address in the AdblockPlus.org domain regularly to retrieve the list of active notifications.}} {{s64 This request makes your IP address visible to the website and also sends some general information like your Adblock Plus version, browser and browser version.}} {{s65 This data allows serving different notifications to different clients and is subject to the usual AdblockPlus.org privacy policy.}}

{{s66 Further AdblockPlus.org website requests}}

{{s67 Adblock Plus may make further requests to the AdblockPlus.org website as required, for example if a documentation link is clicked or if the full list of filter subscriptions needs to be downloaded.}} {{s68 These requests are subject to the usual AdblockPlus.org privacy policy.}}

{{s69 Data retention}}

{{s72 Adblock Plus stores some data in the Firefox profile on your computer.}} {{s73 Adblock Plus never transmits this data to any servers, but other extensions and services, such as}} {{s74 Firefox Sync}}{{s75 , may do so.}} {{s76 Most of the data (your preferences, filter subscriptions and custom filters) is unobjectionable privacy-wise.}} {{s77 However, filter hit statistics and recent issue reports could be used to reconstruct your browsing history.}} {{s78 Adblock Plus treats this information identically to how history data is treated by the browser:}} {{s79 this data isn't stored if you are using Private Browsing mode and is removed if you choose to clear your browsing history.}}
