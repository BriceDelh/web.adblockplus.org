{{s1 The Adblock Plus project consists of multiple repositories.}} {{s2 The modules page has a good overview of the most important ones.}}

{{s3 Getting the source code from the Mercurial repositories}}

{{s4 The Adblock Plus source code is hosted on our Mercurial server.}} {{s5 You can see a list of all repositories through the}} {{s6 web interface}}{{s7 .}}

{{s8 Here's how you clone the Adblock Plus for Firefox repository:}}

hg clone https://hg.adblockplus.org/adblockplus/

{{s9 Here's how you get the source code for a specific version (e.g.}} Adblock Plus 2.3.1{{s10 ):}}

hg update -r 2.3.1

{{s16 Getting the source code from the GitHub mirrors}}

{{s17 If you prefer to use Git, you can - we have GitHub mirrors for all our public repositories.}}

{{s18 Here's how you clone the Adblock Plus for Firefox repository from GitHub:}}

git clone https://github.com/adblockplus/adblockplus

{{s19 Here's how you get the source code for a specific version (e.g.}} Adblock Plus 2.3.1{{s20 ):}}

git checkout 2.3.1

{{s21 Please note: We are currently using Mercurial subrepositories heavily, but are unable to mirror these as Git submodules.}} {{s22 That means that you need to check the}} {{s23 .hgsub file of each repository and clone all dependencies manually.}} {{s24 This won't be necessary anymore once we've}} {{s25 moved away from subrepositories.}}
