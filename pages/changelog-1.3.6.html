title=Detailed changelog for Adblock Plus 1.3.6

{{s1 The following lists the changes compared to Adblock Plus 1.3.5.}}

  • {{s2 "Recommend us on Facebook" button can now be easily hidden (click the cross next to it).}}
  • {{s3 New translations: Esperanto, Upper Sorbian}}
  • {{s4 Improved display of web addresses in Adblock Plus user interface for right-to-left languages (Hebrew, Arabic).}}
  • {{s5 Fixed: Web interfaces of some routers expect Connection header to be always last and are broken by Do Not Track header (bug 23845).}}
  • {{s6 Fixed: Adblock Plus user interface is broken in Thunderbird and SeaMonkey Mail if a non-default window layout is used.}}
  • {{s7 Fixed: Flash is broken in outdated Firefox 3.6.x versions (up to Firefox 3.6.12).}}
  • {{s8 Fixed: Redirect blocking feature has a negative impact on memory management in Firefox (bug 645678).}} {{s9 Side-effect: redirect blocking will no longer work in Firefox Mobile 4 due to}} {{s10 bug 646373}}{{s11 .}}
  • {{s12 Fixed: Issue reporter claims "too many filter subscriptions" even if the subscriptions are disabled.}}
  • {{s13 Fixed: Middle-clicking ABP icon has seemingly no effect if Adblock Plus is disabled on the page.}}
