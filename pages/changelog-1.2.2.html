title=Detailed changelog for Adblock Plus 1.2.2

{{s1 The focus of Adblock Plus 1.2.2 is improving compatibility with Firefox 4, the following lists the changes compared to Adblock Plus 1.2.1.}}

  • {{s2 Updated for asynchronous redirect API in Firefox 4 Beta 4 (bug 546606)}}
  • {{s3 Stopped using <popup> tag for context menus which is transparent in Firefox 4 Beta 3 (bug 578322) — not entirely fixed, still and issue for the toolbar icon context menu}}
  • {{s4 Stopped using nsIDOMNSHTMLAnchorElement interface which no longer exists in Firefox 4 Beta 4 (bug 582303)}}
  • {{s5 Subscription dialog: Made "Add a different subscription" button less prominent by turning it into a link}}
  • {{s6 Subscription dialog: Fixed bad Chinese translation for the "Add a different subscription" button}}
