title=Report an issue

{{s5 Please consider the following points before filing your bug report.}}

{{s6 Some issues aren't Adblock Plus bugs}}

  • {{s7 An advertisement isn't blocked: this is a filters issue, please report it by other means.}}
  • {{s8 A website doesn't work correctly and temporarily disabling Adblock Plus on this site fixes the issue: this is a filters issue, please report it by other means.}}
  • {{s9 An issue happens before Adblock Plus is installed or persists if it is uninstalled: you have to look elsewhere for an explanation.}} {{s10 In particular, if Adblock Plus fails to install and you are using a supported browser then you are facing a browser issue and the}} {{s12 general troubleshooting instructions}} {{s13 apply. If not using Firefox, see:}} {{s15 Chrome}}{{s16 ,}} {{s18 Opera}}{{s19 ,}} {{s21 Safari}}{{s22 ,}} {{s24 Internet Explorer}}

{{s25 How to report bugs}}

{{s26 You can get support and report your issue in the forum.}}

{{s27 Your bug report should contain all necessary information to reproduce it:}}

  • {{s28 Adblock Plus version, which browser you are using (with version number), which operating system is used}}
  • {{s29 Any configuration details that might be relevant (filter lists used, own filters if any, unusual browser configuration, other extensions)}}
  • {{s30 Exact description of the steps that need to be followed to see the bug}}
  • {{s31 Exact description of what happens and what you think is wrong}}

{{s32 My bug report is no longer there, what happened?}}

{{s33 You came back and your bug report is no longer there?}} {{s34 Maybe the moderators changed the title to make it more meaningful.}} {{s35 Look around in the forum.}}
